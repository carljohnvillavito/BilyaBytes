# Cloud Storage App

A simple, secure, and no-login cloud storage application.

## Prerequisites

- Node.js installed on your machine.

## Setup

1.  Open a terminal in this directory.
2.  Install dependencies:
    ```bash
    npm install
    ```

## Running the App

1.  Start the server:
    ```bash
    npm start
    ```
2.  Open your browser and navigate to:
    `http://localhost:3000`

## Features

- **Drag & Drop Upload**: Upload multiple files easily.
- **Shareable Links**: Get instant links to share your files.
- **No Login Required**: Simple and fast.
- **Persistent Storage**: Files are saved in the `uploads` folder.
